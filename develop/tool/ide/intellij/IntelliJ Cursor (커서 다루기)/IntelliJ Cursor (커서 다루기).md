# 커서 단어별로 이동

- Ctrl + 좌우화살표

# 커서 단어별로 블록지정

- Ctrl + Shift + 좌우화살표

# 커서 블록별로 블록지정

- 어느 한 태그 및 그 하위태그들이나 쿼리를 한번에 블록지정하고싶을때 해당태그에서 사용
    - 예를들면 div태그에서 사용시 div태그 및 그 하위태그 한번에 블록지정됨
- Alt + Shift + Up

# 들여쓰기 및 들여쓰기 해제

- 들여쓰기 : Tab
- 들여쓰기 해제 : Shift + Tab

# 멀티커서

- 마우스휠버튼 + 드래그
- 혹은 Ctrl 두번+위아래 방향키

# 특정 라인들만 멀티커서

- Alt+Shift+좌클릭

# 기본 드래그 멀티커서 ON/OFF

- Alt+Shift+Insert

# 태그/브라켓 시작, 마지막으로 이동

- Ctrl + { : 시작 지점으로 이동
- Ctrl + } : 마지막 지점으로 이동
