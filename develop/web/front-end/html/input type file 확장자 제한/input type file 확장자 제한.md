```html
<input type="file" accept="파일 확장자 | audio/* | video/* | image/* | 미디어 타입">
```

- 파일 확장자
    - .으로 시작하는 확장자 (ex : .png, .pdf .....)
- audio/*
    - 모든 타입의 오디오 파일
- video/*
    - 모든 타입의 비디오 파일
- image/*
    - 모든 타입의 이미지 파일
- 미디어 타입
    - 매개변수를 가지지 않는 유효한 미디어 타입

> accept속성은 input type이 file일때만 유효한 속성
