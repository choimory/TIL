# 앞으로가기, 뒤로가기

- `Cmd + [, ]`

# Delete 키

- `fn + Backspace`

# Home, end

- `fn + 좌우 방향키`

# Page up, Page down

- `fn + 위아래 방향키`

# 브라우저 탭 이동

- `fn + ctrl + 위아래 방향키`

# IntelliJ 탭 이동

- `cmd + 위아래 방향키`

# IntelliJ 주석

- 라인 주석: `cmd + /`
- 블록 주석: `shift + ctrl + /`

# 커서 이동

- 단어 단위로 이동 : `Opt + 좌우방향키`
- 라인 처음과 끝으로 이동 : `cmd + 좌우 방향키`

# 한영키

- `Caps Lock` (사실 Caps Lock 버튼 위치에 한영키가 있는것)

# ` (백쿼트)

# 바탕화면 보기

- `Cmd + F3` 혹은 `Fn + F11`
